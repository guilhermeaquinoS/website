# Personal Website — Guilherme de Aquino

This repository contains the source code for my personal academic and professional website.

The website presents my background, research activities, publications, and contact information, with a focus on fluid dynamics, computational fluid dynamics (CFD), experiments, and data-driven analysis.

---

## Overview

This website is implemented as a **static website** using standard web technologies:

- HTML  
- CSS  
- Minimal JavaScript (for layout, navigation, and small interactions)

The website is hosted using **GitHub Pages**.

---

## Implementation and Authorship

The website was **designed and implemented by me**, including its structure, layout, styling, and all written content.

During development, I used **ChatGPT as an assistance tool** to:
- Refine and improve written content
- Debug and optimize HTML/CSS code
- Improve layout consistency and responsiveness
- Apply basic accessibility and SEO best practices

All design decisions, technical choices, and final implementations were made, reviewed, and validated by me.
